Getting Started

Install Node.js and run npm install systeminformation. Then start the server with node server.js.

Features Overview

  • Network Monitoring: Monitor performance metrics, traffic, and devices.
  • Intrusion Detection: Set rules and detect anomalies.
  • Alerts: Real-time notifications with mitigation suggestions.
  • Dashboard: Interactive interface with customizable rules.
  • Security Integration: Traffic and attack monitoring.
